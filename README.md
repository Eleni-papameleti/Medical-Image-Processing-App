# 🔬 Medical Image Processing App

Μια διαδραστική εφαρμογή βασισμένη στο **Streamlit** για την επεξεργασία και ανάλυση ιατρικών εικόνων. Η εφαρμογή καλύπτει ένα ευρύ φάσμα τεχνικών, από βασικές τροποποιήσεις απεικόνισης έως προχωρημένη ανακατασκευή τομογραφίας.

## 🌟 Χαρακτηριστικά

Η εφαρμογή χωρίζεται σε 4 βασικές ενότητες:

*   **📊 Κεφάλαιο 1: Μέθοδοι Απεικόνισης**
    *   Windowing (Simple, Broken, Double Window).
    *   Μετασχηματισμοί LUT (Inverse, Logarithmic, Sigmoid, κ.α.).
    *   Εξισορρόπηση Ιστογράμματος (Histogram Equalization, CLAHE).
*   **🔲 Κεφάλαιο 2: Φιλτράρισμα στο Χωρικό Πεδίο**
    *   Φίλτρα εξομάλυνσης (Smoothing), ανίχνευσης ακμών (Sobel, Laplacian) και όξυνσης.
    *   Δυνατότητα χρήσης Custom Kernels.
    *   Ανάλυση θορύβου και προφίλ έντασης pixel.
*   **〰️ Κεφάλαιο 3: Φιλτράρισμα στο Πεδίο Συχνοτήτων**
    *   Φίλτρα Butterworth, Ideal, Gaussian και Exponential.
    *   Αποκατάσταση εικόνας με χρήση φίλτρων Wiener και Power.
    *   Οπτικοποίηση φάσματος FFT.
*   **🔁 Κεφάλαιο 4: Τομογραφική Ανακατασκευή**
    *   Υλοποίηση αλγορίθμων FBP (Filtered Back Projection) και ART (SART).
    *   Δημιουργία και απεικόνιση Sinograms.

## 🛠️ Τεχνολογίες & Βιβλιοθήκες
Το project χρησιμοποιεί τις εξής βιβλιοθήκες:
- `streamlit` (UI)
- `numpy` & `scipy` (Μαθηματικοί υπολογισμοί)
- `matplotlib` (Οπτικοποίηση)
- `Pillow` & `pydicom` (Διαχείριση εικόνων & αρχείων DICOM)
- `scikit-image` (Αλγόριθμοι Radon/Iradon)
- `opencv-python` (Επεξεργασία εικόνας)
